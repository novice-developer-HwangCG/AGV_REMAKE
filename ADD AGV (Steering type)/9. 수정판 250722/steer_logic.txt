------ 수동 모드 ------

1. 거리 값 없고 회전 값 없을 때 전진 후진
- rotate는 항상 0으로 두어야 함

2. 거리 값 없고 회전 값 있을 때 회전
- speed는 항상 0으로 두어야 함

3. 거리 값 있고 회전 값 없을 때
- rotate는 항상 0으로 두고 수동 모드를 한 번 더 눌러서 수동 모드임을 알리고 거리 값을 전달
- 다음 speed 값을 준 후 전진 명령

------ 자동 모드 ------

1. 거리 값이 없다면 라인트레이싱
- 카메라가 선 중앙 또는 검은색 선이 보이도록 위치 해야함

2. 거리 값이 있을 경우 거리 값 만큼 라인 트레이싱

+ 거리 값 기반 주행 주행 중 정지 시 목표 이동 거리에서 누적 이동 거리(엔코더)를 뺀 남은 거리 값 저장
  재주행 명령 시 남은 거리 값 기반으로 하여 주행 시작

  수동+자동 모드 동일 플로우 지님 단 수동모드는 테스트를 해보지 못함

남은 거리 값 기반 플로우가 불안정할 시 초기화 명령을 준 후 다시 거리 값을 전달후 주행 명령을 내리면 됨
